# ComputerSystemsAndDesignLab
Computer Systems And Design Lab; This revolves around working with the official Nand2Tetris Course from https://www.nand2tetris.org/


# Hardware
### Project 1: Boolean Logic
To build the basic logic gates, followed with multiplexers and de-multiplexers.

### Project 2: Boolean Arithmetic
To build a basic ALU unit using standard adders

### Project 3: Memory
To understand the Flip-flop unit, which fuels the concept of memory storage, and implement various groups of storage units(based on their storage capacities) like registers,RAM and program-counters.

### Project 4: Assembly Language
This project involves an introduction to *low-level assembly language*. Here we work with an assembly language called *Hack* and proceed with understanding its instruction set architecture.

### Project 5: Computer
Here, we would construct a full-fledged computer that is capable of executing binary machine codes and produce the desired outputs.

### Project 6: Assembler
To Implement a machine(*assembler*) that can convert Hack assembly code into corresponding executable binary code
